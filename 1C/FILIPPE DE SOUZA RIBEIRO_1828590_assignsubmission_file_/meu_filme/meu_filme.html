Projeto X

Três amigos de colégio planejam uma festa inesquecível que entrará para a história na tentativa de ficarem famosos. A notícia se espalha rapidamente e tudo foge ao controle quando os imprevistos começam a acontecer.

Trailer do filme

O filme foi dirigido pelo diretor Nima Nourizadeh.

Em 16 de março de 2012

Projeto X é um filme icônico, que mopstra as loucuras que uma festas organizada pro 3 jovens pode causar.

Alguns motivos para você assistir o filme!!!

  • Você se divertira muito.
  • Conhecerá a lucura dos jovens.
  • Poderá se reunir com amigos para rirem juntos.
  • É ótimo para se assistir em um "date" por ser descontraido.
  • Você aprenderá a dar uma boa festa.
